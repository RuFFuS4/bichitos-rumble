Logical state → clip · speed · loop

State Clip Speed ⤴ Play Status
Each row owns its own clip + speed + loop. Changing the clip auto-plays the new selection. Changing speed live-adjusts the running clip without restarting it (no flicker while you type). Loop is the EXPORT setting only — the lab preview always loops so one-shot clips (abilities, victory, defeat, fall) don't freeze at the last frame; the checkbox value ships in the patch for game runtime. Play replays whatever is currently configured. Apply rebuilds the skeletal animator so the resolver runs again — only useful when you've left a state on Auto and want to confirm what it picks. Reset clears every session edit for the selected critter and reloads its authored baseline.
