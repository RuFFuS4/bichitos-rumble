# Bichitos Rumble

A chaotic arena brawler where big-headed critters fight inside a circular arena that progressively collapses. Knock your rivals into the void!

Built for the **2026 Vibe Coding Game Jam**.

## Play

Visit the live version (coming soon) or run locally:

```bash
npm install
npm run dev
```

## Controls

| Key | Action |
|-----|--------|
| WASD / Arrow Keys | Move |
| Space | Headbutt |
| J | Ability 1 |
| K | Ability 2 |
| R | Restart (after match ends) |

## Tech Stack

- TypeScript + Vite
- Three.js for 3D rendering
- No backend required for the prototype
- Deployed on Vercel
